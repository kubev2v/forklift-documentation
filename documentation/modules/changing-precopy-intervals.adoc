= Changing precopy intervals for warm migration

[role="_abstract"]
You can change the snapshot interval by patching the `ForkliftController` custom resource (CR).

.Procedure

* Patch the `ForkliftController` CR:
+
[source,terminal,subs="attributes+"]
----
$ {oc} patch forkliftcontroller/<forklift-controller> -n {namespace} -p '{"spec": {"controller_precopy_interval": <60>}}' --type=merge
----
+
where:

The value after `"controller_precopy_interval"`, which is `<60>` in the preceding code block::
Specifies the precopy interval in minutes. The default value is `60`.
+
You do not need to restart the `forklift-controller` pod.
