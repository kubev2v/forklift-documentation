= OpenStack prerequisites

[role="_abstract"]
{osp} migrations require a compatible version. MTV 2.6 and later support token authentication and application credential authentication in addition to standard credentials. These methods work with the CLI like standard migrations, differing only in `Secret` manifest preparation.
