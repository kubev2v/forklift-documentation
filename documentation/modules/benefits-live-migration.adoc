= Benefits of live migration

[role="_abstract"]
Live migration significantly reduces downtime, enabling migrations with minimal service disruption. It supports VM migration between {virt} clusters and namespaces, eliminates scheduled downtime during maintenance, enables dynamic infrastructure management, and balances workloads across clusters.
