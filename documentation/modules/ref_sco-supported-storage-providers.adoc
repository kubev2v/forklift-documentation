= Supported storage providers

[role="_abstract"]
Use any of these storage providers with storage copy offload:

* Hitachi Vantara
* NetApp ONTAP
* Pure Storage FlashArray
* Dell PowerMax
* Dell PowerFlex
* Dell PowerStore
* HPE 3PAR
* HPE Primera
* Infinidat InfiniBox
* IBM Flashsystem
