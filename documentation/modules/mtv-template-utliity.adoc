= {project-short} template utility for {vmw} VM names

[role="_abstract"]
You can use the template utility of {project-first} to generate names for your virtual machines (VMs). Using names generated by these methods reduce the possibilities of problems working with your VMs after their migration to {virt}.

The utility includes string and mathematical models.

[id="mtv-template-utility-usage_{context}"]
== Usage

The code block that follows gives the usage of the template utility used for generating {vmw} VM names:

.Template utility used for generating {vmw} VM names
[source,terminal]
----
import (
    "text/template"
    "github.com/kubev2v/forklift/pkg/templateutil"
)

// Add template functions to a FuncMap
funcMap := template.FuncMap{}
funcMap = templateutil.AddStringFuncs(funcMap)
funcMap = templateutil.AddMathFuncs(funcMap)

// Or use all functions at once
funcMap = templateutil.AddTemplateFuncs(funcMap)

// Or use the ExecuteTemplate helper
result, err := templateutil.ExecuteTemplate("Hello, {{ upper .Name }}!", data)
----

[id="mtv-template-utility-template-functions_{context}"]
== Available template functions

The tables that follow describe that string and mathematical functions that you can use in templates that rename VMs for use with {project-short}:

[cols="1,1,1",options="header"]
.String functions

|===
| Function
| Description
| Example

| `lower`
| Converts a string to lowercase
| `{{ lower "TEXT" }}` → `text`

| `upper`
| Converts a string to uppercase
| `{{ upper "text" }}` → `TEXT`

| `contains`	
| Checks if a string contains a specific substring
| `{{ contains "hello" "lo" }}` → `true`

| `replace`
| Replaces occurrences in a string
| `{{"I Am Henry VIII" \| replace "{nbsp}" "-"}} → I-Am-Henry-VIII`

| trim
| Removes whitespace from both ends of a string
| `{{ trim "  text  " }}`→ `text`

| trimAll
| Removes specified characters from both ends of a string
| `{{ trimAll "$" "$5.00$" }}` → `5.00`

| trimSuffix
| Removes the specified suffix from the string, if it is present
| `{{ trimSuffix ".go" "file.go" }}` → `file`

| trimPrefix	Removes prefix if present	{{ trimPrefix "go." "go.file" }} → file
|
|

| title	Converts to title case	{{ title "hello world" }} → Hello World
|
|

| untitle	Converts to lowercase	{{ untitle "Hello World" }} → hello world
|
|

| repeat	Repeats string n times	{{ repeat 3 "abc" }} → abcabcabc
|
|

| substr	Extracts substring from start to end	{{ substr 1 4 "abcdef" }} → bcd
|
|

| nospace	Removes all whitespace	{{ nospace "a b  c" }} → abc
|
|

| trunc	Truncates string to specified length	{{ trunc 3 "abcdef" }} → abc
|
|

| initials	Extracts first letter of each word	{{ initials "John Doe" }} → JD
|
|

| hasPrefix	Checks if string starts with prefix	{{ hasPrefix "go" "golang" }} → true
|
|
 
| hasSuffix	Checks if string ends with suffix	{{ hasSuffix "ing" "coding" }} → true
|
|

| mustRegexReplaceAll	Replaces matches using regex with submatch expansion	{{ mustRegexReplaceAll "a(x*)b" "-ab-axxb-" "${1}W" }} → -W-xxW-
|
|===
