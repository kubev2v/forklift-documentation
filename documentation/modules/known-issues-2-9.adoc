= Known issues

{project-first} 2.9 has the following known issues:

.Persistent volume claim name template does not support the `shared` field
The PVC name template in {project-short} user interface does not support the `shared` field to name volumes shared by multiple virtual machines. link:https://issues.redhat.com/browse/MTV-2721[(MTV-2721)]

.Vmware guest virtual machines running Red Hat Enterprise Linux v6.0 cannot boot in {project-short}
Vmware guest VMs running RHEL v6.0 cannot boot into OS because the `virtio` disk of RHEL 6.0 requires `virtio-transitional` device. link:https://issues.redhat.com/browse/MTV-1895[(MTV-1895)]

For a complete list of all known issues in this release, see the list of link:https://issues.redhat.com/issues/?filter=12472621[Known Issues] in Jira.
