[id="live-migration-limitations_{context}"]
= Limitations of live migration

[role="_abstract"]
Live migration works only between {virt} clusters or namespaces, not with other source providers. It does not establish cluster connectivity or migrate non-VM resources like services, routes, or application components needed for post-migration application availability.
