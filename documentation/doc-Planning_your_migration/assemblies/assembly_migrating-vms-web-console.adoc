= Migrating virtual machines by using the {ocp} web console

//:context: migrating-vms-web

[role="_abstract"]
Use the {project-short} user interface to migrate virtual machines (VMs) from {vmw} vSphere, {rhv-full} ({rhv-short}), {osp}, Open Virtual Appliances (OVAs) that were created by {vmw} vSphere, or {virt} clusters. For all migrations, you specify the source provider, the destination provider, and the migration plan. The specific procedures vary per provider.

[IMPORTANT]
====
You must ensure that all prerequisites are met.

{vmw} only: You must have the minimal set of {vmw} privileges.

{vmw} only: Creating a {vmw} Virtual Disk Development Kit (VDDK) image will increase migration speed.
====
